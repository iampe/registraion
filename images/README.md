# Images Directory

This directory should contain your website images.

## Required Images:

1. **payment-qr.png** - Your UPI payment QR code
   - Size: 300x300px recommended
   - Format: PNG with transparent background preferred
   - This QR code will be displayed on all course detail pages

## How to Add Images:

1. Create your UPI payment QR code using any UPI app (Google Pay, PhonePe, etc.)
2. Take a screenshot and crop it to just the QR code
3. Save it as `payment-qr.png` in this directory
4. The website will automatically display it on the course details pages

## Note:

- Course images are currently loaded from Unsplash CDN
- You can replace these with your own images by updating the URLs in `js/data.js`
- Recommended course image size: 800x600px
- Recommended banner image size: 1200x600px
